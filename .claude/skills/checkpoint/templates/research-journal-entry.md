# Research Journal Entry Format

Append to `quality_reports/research_journal.md` whenever an agent completes work. Append only -- never overwrite existing entries. Only add an entry if agent work happened this session.

### YYYY-MM-DD HH:MM -- [Agent Name]
**Phase:** [Discovery/Strategy/Execution/Peer Review/Presentation]
**Target:** [file or topic]
**Score:** [XX/100 or PASS/FAIL or N/A]
**Verdict:** [one line -- key finding or decision]
**Report:** [path to full report]
