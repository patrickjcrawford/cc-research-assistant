# Session Report Entry Format

Append to `SESSION_REPORT.md` at end of session or before context compression. Append only -- never overwrite existing entries.

## YYYY-MM-DD HH:MM -- [Brief Title]

**Operations:**
- [Scripts run, files created/modified/deleted]

**Decisions:**
- [Choice made] -- [rationale]

**Results:**
- [Key findings, outputs produced]

**Commits:**
- `[hash]` [commit message]

**Status:**
- Done: [what's complete]
- Pending: [what remains]
